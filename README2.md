# Multi-PDF-RAG-Pipeline

## Overview
This project implements a Retrieval-Augmented Generation (RAG) pipeline using multiple PDF files.

## Features
- Multi PDF support
- Text chunking
- Embedding generation
- ChromaDB vector database
- Semantic similarity search
- LLM based response generation

## Technologies Used
- Python
- LangChain
- ChromaDB
- Sentence Transformers
- Transformers

## Workflow
PDF → Chunking → Embedding → Vector DB → Retrieval → LLM Response
